# 基本機能テスト

このファイルは、Markdown基本構文とGitHub Flavored Markdown (GFM)の機能が正しく表示されるかをテストします。

---

## 1. 見出し（Headings）

# 見出しレベル1（H1）
## 見出しレベル2（H2）
### 見出しレベル3（H3）
#### 見出しレベル4（H4）
##### 見出しレベル5（H5）
###### 見出しレベル6（H6）

**期待される表示**:
- H1とH2は下線あり
- H1が最も大きく、H6が最も小さい
- すべての見出しが太字

---

## 2. 段落（Paragraphs）

これは普通の段落です。複数の文を含むことができます。段落の後には空白行を入れます。

これは2つ目の段落です。段落と段落の間には空行が必要です。
改行だけでは新しい段落になりません。

段落内で改行したい場合は、
行末にスペースを2つ入れます。
これで改行されます。

**期待される表示**:
- 段落間に適切な余白がある
- 行末スペース2つで改行される

---

## 3. 強調（Emphasis）

### 太字（Bold）

**太字のテキスト（アスタリスク2つ）**

__太字のテキスト（アンダースコア2つ）__

### 斜体（Italic）

*斜体のテキスト（アスタリスク1つ）*

_斜体のテキスト（アンダースコア1つ）_

### 太字+斜体

***太字かつ斜体（アスタリスク3つ）***

___太字かつ斜体（アンダースコア3つ）___

### 打ち消し線

~~これは打ち消された文字です~~

通常のテキストと~~打ち消し~~が混在しています。

**期待される表示**:
- 太字：font-weight 600
- 斜体：font-style italic
- 太字+斜体：両方適用
- 打ち消し線：横線が引かれる

---

## 4. リスト（Lists）

### 番号なしリスト（Unordered List）

- 項目1
- 項目2
- 項目3
  - ネスト項目3-1
  - ネスト項目3-2
    - さらにネスト3-2-1
- 項目4

### 番号付きリスト（Ordered List）

1. 最初の項目
2. 2番目の項目
3. 3番目の項目
   1. ネスト項目3-1
   2. ネスト項目3-2
4. 4番目の項目

### 混合リスト

1. 番号付き項目1
   - 番号なし項目1-1
   - 番号なし項目1-2
2. 番号付き項目2
   1. 番号付きネスト2-1
   2. 番号付きネスト2-2

**期待される表示**:
- 適切なインデント
- 番号なしリストには・（箇条書き記号）
- 番号付きリストには1.、2.、3.

## 5. タスクリスト（GFM）

### 基本的なタスクリスト

- [x] 完了したタスク1
- [x] 完了したタスク2
- [ ] 未完了のタスク1
- [ ] 未完了のタスク2
- [ ] 未完了のタスク3

### ネストしたタスクリスト

- [x] メインタスク1
  - [x] サブタスク1-1
  - [x] サブタスク1-2
  - [ ] サブタスク1-3
- [ ] メインタスク2
  - [ ] サブタスク2-1
  - [ ] サブタスク2-2

### テキストを含むタスクリスト

- [x] **太字のタスク**
- [ ] *斜体のタスク*
- [x] `コード`を含むタスク
- [ ] [リンク](https://example.com)を含むタスク

### 打ち消し線と組み合わせ

- [x] ~~完了した項目~~
- [ ] 進行中の項目
- [x] ~~キャンセルされた項目~~

**期待される表示**:
- チェックボックスが表示される
- チェック済みタスクにチェックマークが入っている
- 未チェックタスクは空のチェックボックス
- インデントが正しく表示される
- リストマーカー（・）が表示されない

## 6. リンク（Links）

### 基本的なリンク

[Googleへのリンク](https://www.google.com)

[タイトル付きリンク](https://www.example.com "これはタイトルです")

### 自動リンク（GFM）

https://www.google.com

http://example.com

test@example.com

### 内部リンク（同じページ内）

[見出しへのリンク](#1-見出しheadings)

### 強調とリンクの組み合わせ

**[太字のリンク](https://example.com)**

*[斜体のリンク](https://example.com)*

`[コード内のリンク構文](https://example.com)`

**期待される表示**:
- 青色のリンク（#0969da）
- ホバー時に下線
- 外部リンクは新しいタブで開く（target="_blank"）
- URLとメールアドレスが自動的にリンク化

## 8. 引用（Blockquotes）

> これは引用文です。
> 複数行にわたることができます。

> #### 引用内に見出し
>
> 引用の中に**太字**や*斜体*を含むこともできます。
>
> - リストも含められます
> - 2つ目の項目

### ネストした引用

> レベル1の引用
>
> > レベル2の引用（ネスト）
> >
> > > レベル3の引用

**期待される表示**:
- 左側に縦の線（0.25em、#d0d7de色）
- テキスト色がやや薄い（#57606a）
- ネストごとにインデント

## 9. コード（Code）

### インラインコード

これは`inline code`の例です。`const x = 10;`のようなコードを文中に含めます。

### コードブロック（言語指定あり）

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

```python
def example():
    return "Python code"
```

### コードブロック（言語指定なし）

```
plain text code block
no syntax highlighting
```

**期待される表示**:
- インラインコード：薄いグレー背景、等幅フォント
- コードブロック：薄いグレー背景、パディングあり、シンタックスハイライト

## 10. テーブル（GFM）

### 基本的なテーブル

| ヘッダー1 | ヘッダー2 | ヘッダー3 |
|----------|----------|----------|
| セル1    | セル2    | セル3    |
| データA  | データB  | データC  |

### テキスト揃え

| 左揃え | 中央揃え | 右揃え |
|:-------|:--------:|-------:|
| Left   | Center   | Right  |
| 左     | 中央     | 右     |
| A      | B        | C      |

### 複雑なコンテンツを含むテーブル

| 項目 | 説明 | 状態 |
|------|------|------|
| **太字** | これは太字です | ✅ |
| *斜体* | これは斜体です | ⚠️ |
| `コード` | `inline code` | 🔵 |
| [リンク](https://example.com) | 外部リンク | 🔗 |

### 日本語を含むテーブル

| クラス名 | 説明 | ファイル |
|---------|------|---------|
| CCNode | すべてのノードの基底クラス | Cocos2d-x |
| CCSprite | 画像表示用クラス | Cocos2d-x |
| CCLayer | レイヤークラス | Cocos2d-x |

### 空セルを含むテーブル

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
|     | 空  |     |
| X   |     | Z   |

**期待される表示**:
- ヘッダー行が灰色の背景になっている
- 罫線が表示されている
- セルに適切なパディングがある
- 2行目ごとに背景色が交互になっている（ゼブラストライプ）
- テキスト揃え（左/中央/右）が正しく機能している
- テーブル内の強調、リンク、コードが正しく表示される

## 11. 水平線（Horizontal Rules）

3つのハイフン:

---

3つのアスタリスク:

***

3つのアンダースコア:

___

**期待される表示**:
- 太い水平線（0.25em）
- グレー色（#d0d7de）
- 上下に余白（24px）
