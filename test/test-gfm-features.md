# GitHub Flavored Markdown (GFM) 機能テスト

このファイルは、GFMの主要機能（タスクリスト、テーブル）が正しく表示されるかをテストします。

---

## テスト1: タスクリスト

### 基本的なタスクリスト

- [x] 完了したタスク1
- [x] 完了したタスク2
- [ ] 未完了のタスク1
- [ ] 未完了のタスク2
- [ ] 未完了のタスク3

### ネストしたタスクリスト

- [x] メインタスク1
  - [x] サブタスク1-1
  - [x] サブタスク1-2
  - [ ] サブタスク1-3
- [ ] メインタスク2
  - [ ] サブタスク2-1
  - [ ] サブタスク2-2

### テキストを含むタスクリスト

- [x] **太字のタスク**
- [ ] *斜体のタスク*
- [x] `コード`を含むタスク
- [ ] [リンク](https://example.com)を含むタスク

### 期待される表示

- ✅ チェックボックスが表示される
- ✅ チェック済みタスクにチェックマークが入っている
- ✅ 未チェックタスクは空のチェックボックス
- ✅ インデントが正しく表示される
- ✅ リストマーカー（・）が表示されない

---

## テスト2: テーブル

### 基本的なテーブル

| ヘッダー1 | ヘッダー2 | ヘッダー3 |
|----------|----------|----------|
| セル1    | セル2    | セル3    |
| データA  | データB  | データC  |

### テキスト揃え

| 左揃え | 中央揃え | 右揃え |
|:-------|:--------:|-------:|
| Left   | Center   | Right  |
| 左     | 中央     | 右     |
| A      | B        | C      |

### 複雑なコンテンツを含むテーブル

| 項目 | 説明 | 状態 |
|------|------|------|
| **太字** | これは太字です | ✅ |
| *斜体* | これは斜体です | ⚠️ |
| `コード` | `inline code` | 🔵 |
| [リンク](https://example.com) | 外部リンク | 🔗 |

### 日本語を含むテーブル

| クラス名 | 説明 | ファイル |
|---------|------|---------|
| CCNode | すべてのノードの基底クラス | Cocos2d-x |
| CCSprite | 画像表示用クラス | Cocos2d-x |
| CCLayer | レイヤークラス | Cocos2d-x |

### 長いテキストを含むテーブル

| 項目 | 詳細説明 |
|------|---------|
| 概要 | この項目は非常に長いテキストを含んでいます。テーブルセルが適切に伸縮するかをテストするためのサンプルテキストです。 |
| 機能 | GitHub Flavored Markdownのテーブル機能をテストしています。複数の列を持ち、各セルに異なる長さのテキストが入っています。 |

### 空セルを含むテーブル

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
|     | 空  |     |
| X   |     | Z   |

### 期待される表示

- ✅ ヘッダー行が灰色の背景になっている
- ✅ 罫線が表示されている
- ✅ セルに適切なパディングがある
- ✅ 2行目ごとに背景色が交互になっている（ゼブラストライプ）
- ✅ テキスト揃え（左/中央/右）が正しく機能している
- ✅ テーブル内の強調、リンク、コードが正しく表示される

## テスト3: 打ち消し線

### 基本的な打ち消し線

~~これは打ち消された文字です~~

通常のテキストと~~打ち消し~~が混在しています。

### リストと組み合わせ

- ~~完了した項目~~
- 進行中の項目
- ~~キャンセルされた項目~~

### 期待される表示

- ✅ 打ち消し線が表示される
- ✅ 文字が読める程度の線の太さ

## 総合確認チェックリスト

F12デベロッパーツールで以下を確認してください：

### タスクリスト確認

```javascript
// Consoleで実行
document.querySelectorAll('input[type="checkbox"]').length
// → チェックボックスの数が表示される（10以上あるはず）
```

### テーブル確認

```javascript
// Consoleで実行
document.querySelectorAll('table').length
// → テーブルの数が表示される（5以上あるはず）
```

### 打ち消し線確認

```javascript
// Consoleで実行
document.querySelectorAll('del').length
// → 打ち消し線の数が表示される（3以上あるはず）
```
