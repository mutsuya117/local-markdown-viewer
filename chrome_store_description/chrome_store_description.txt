【概要】

Local Markdown Viewer は、Chromeで開いたローカルのMarkdown(.md / .markdown)ファイルを、GitHubスタイルで美しく表示する拡張機能です。
見出し（h1～h6）から自動的に目次を生成し、ページ内リンクで素早く移動できます。
GitHub Flavored Markdown (GFM) に完全対応し、テーブル、タスクリスト、シンタックスハイライト、Mermaidダイアグラム、数式表示など、豊富な機能を提供します。ダークモード切り替え、印刷機能、HTMLエクスポート機能も搭載し、資料共有や配布にも最適です。
設定画面はなく、インストールするだけで使えるシンプルな設計です。


【主な機能】

■ Markdown表示機能
・GitHubスタイルで表示、GitHub Flavored Markdown (GFM) 対応
・140以上のプログラミング言語のシンタックスハイライトに対応
・自動目次生成（h1～h6）、目次幅のリサイズ機能、レスポンシブデザイン

■ Mermaidダイアグラム
・フローチャート、シーケンス図、ガントチャートなど多数のダイアグラムに対応
・コードブロックで自動描画

■ 数式表示（KaTeX）
・LaTeX形式の数式をサポート（インライン・ディスプレイ数式）
・ワンクリックでON/OFF切り替え可能

■ ダークモード
・ワンクリックでライト/ダークモードを切り替え
・設定は自動保存

■ 印刷機能
・ワンクリックで印刷、目次とボタンは自動的に非表示
・現在のモード（ライト/ダーク）を維持

■ HTMLエクスポート機能
・ワンクリックでスタンドアロンHTMLファイルとしてダウンロード
・ローカル画像を自動的にBase64として埋め込み（PNG, JPEG, GIF, WebP, SVG, BMP対応）
・拡張機能がなくてもどのブラウザでも閲覧可能、資料共有に最適

■ 多言語対応
・英語、日本語、中国語簡体字に対応

■ セキュリティ
・プライバシー保護 - ファイル内容を外部に送信しません


【動作範囲と制限】

■ 対象
・file:// プロトコルで開いたMarkdown（.md / .markdown）ファイル
・ローカルに保存されているMarkdownファイル

■ 非対象
・Web上にあるMarkdown表示ページ（GitHub、GitLab、Bitbucketなど）
・リモートのGitリポジトリのプレビューページ
・その他のファイル形式（.txt、.html など）

■ 動作の仕組み
・拡張機能は表示中のページ内だけで処理を行います
・Markdownのテキストを読み取り、HTMLに変換して表示します
・ファイルの内容を外部サーバーに送信することはありません
・Markdownファイル内に記載された画像リンクがある場合、その画像を表示します
 （記載されていないファイルに勝手にアクセスすることはありません）


【権限について】

■ 必要な設定
「ファイルのURLへのアクセスを許可する」
この設定を有効にすると、拡張機能がローカルファイル（file://）を読み取れるようになります。

■ 権限の使用目的
・ローカルのMarkdownファイルを読み取り、HTML形式に変換して表示するためだけに使用します
・HTMLエクスポート機能で、Markdown内に記載された画像ファイルを読み取り、Base64として埋め込むために使用します
・ファイルの内容を変更・削除・上書きすることは一切ありません
・読み取り専用で、表示のみを行います

■ プライバシーについて
・データ収集なし - ユーザーデータや閲覧履歴を収集しません
・外部送信なし - ファイル内容を外部サーバーに送信しません
・トラッキングなし - 広告やアクセス解析は一切行いません
・完全ローカル処理 - すべての処理はブラウザ内で完結します


【この拡張機能が向いている人】

・READMEや技術仕様書などをMarkdownで書いていて、ブラウザでも読みやすく見たい人
・社内・社外に公開しないドキュメントをローカルで安全に閲覧したい人
・Markdown資料を美しいHTMLファイルとして配布・共有したい人
・設定を増やしたくなく、シンプルさを優先したい人


【よくある質問】

Q. GitHubのMarkdownページには効きますか？
A. いいえ。この拡張機能は「ローカルで開いた.md/.markdownファイル」だけを整形します。
   Web上のリモートページには一切干渉しません。

Q. データはどこかに送られますか？
A. いいえ。ページ内で完結しており、外部に送信しません。
   すべての処理はブラウザ内で行われます。

Q. ファイルを編集することはできますか？
A. いいえ。この拡張機能は読み取り専用です。
   ファイルの内容を変更・削除・上書きすることは一切ありません。

Q. 設定方法がわかりません
A. インストール時に自動的にセットアップガイドページが開き、
   設定方法をステップバイステップで案内します。ガイドに従って
   「ファイルのURLへのアクセスを許可する」を有効にするだけで使用できます。

Q. 対応している言語は何ですか？
A. 英語、日本語、中国語簡体字に対応しています。
   Chromeブラウザの言語設定に基づいて、自動的に表示言語が選択されます。
